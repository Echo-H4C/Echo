# 1. 정규 표현식 (RegExp)

특정한 규칙을 가진 문자열의 집합을 표현하기 위해 사용하는 형식 언어

# 2. 기본 문법

. : 임의의 한 문자와 일치 (줄바꿈 제외) 

예시) a.b => aab, acb, abb 

^ : 문자열의 시작을 의미 

예시) ^qq => qq로 시작하는 문장을 의미 

$ : 문자열의 끝을 의미

예시) dd$ => dd로 끝나는 문장을 의미 

\* : 앞 문자가 0번 이상 반복

예시) a*c => ac, aac, aaac 등

\+ : 앞 문자가 1번 이상 반복

예시) k+c => kkc, kkkc 

?? : 앞 문자가 0번 또는 1번 존재

예시) a?b => b 또는 ab

[] : 괄호 안의 문자 중 하나와 일치 

예시) [abc] => a 또는 b 또는 c
